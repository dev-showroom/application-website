The Famous Snake Game

The iconic and popular game 'Snake', which was on the first first Nokia cell phones, I have recreated it here in Python.

DEMO

Rules of the game

The player controls the snake with the arrow keys. The aim of the game is to eat food with the snake and make it grow in the process. The more food the snake eats, the higher the score. If it collides with the walls or itself, the game ends.
