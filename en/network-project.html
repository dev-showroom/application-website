Network project

Goals

  • 1. Creation of an isolated network to protect against possible threats from the existing main network
  • 2. Implementation of VLANs for network segmentation and optimization of broadcast traffic
  • 3. Configuration of various mechanisms of the firewall to defense against threats
  • 4. Set up Wake on LAN and remote connections to enable local access to all end devices
